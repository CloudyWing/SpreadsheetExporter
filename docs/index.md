# SpreadsheetExporter

用於匯出 Excel 試算表的函式庫，透過範本系統定義表格版面配置。
核心負責描述 `SpreadsheetDocument`、`SheetDefinition` 與各種 template，實際 `.xlsx` 由 ClosedXML renderer 輸出。
